RhoVaporMixtureFromPressureTemperatureIC

Initial condition to set density of vapor mixture from given pressure and temperature using the SinglePhaseFluidProperties and VaporMixtureInterface interface.

    C++ Type:std::vector<VariableName>

    Controllable:No

    Description:Temperature of the mixture [K]

  • fp_vapor_mixtureVapor mixture fluid properties user object name

    C++ Type:UserObjectName

    Controllable:No

    Description:Vapor mixture fluid properties user object name

  • pPressure of the mixture [Pa]

    C++ Type:std::vector<VariableName>

    Controllable:No

    Description:Pressure of the mixture [Pa]

  • variableThe variable this initial condition is supposed to provide values for.

    C++ Type:VariableName

    Controllable:No

    Description:The variable this initial condition is supposed to provide values for.

  • x_secondary_vaporsMass fractions of secondary vapors

    C++ Type:std::vector<VariableName>

    Controllable:No

    Description:Mass fractions of secondary vapors
